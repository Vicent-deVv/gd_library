# GD Project

A python library for interacting with geometry dash data through a simple and intuitive API

## Features

-  Retrieve Geometry Dash levels 
-  Search players (Work in progress)
-  Access level statistics (Work in progress)
-  Decode and encode Geometry Dash data (Work in progress, Decode is already working)
-  Easy-to-use API 
-  Lightweight and fast

## Installation

this library does'nt use many libraries, but it's necessary for proper functioning

```bash
git clone https://github.com/Vicent-deVv/gd_library.git

cd gd_library

pip install -r requirements.txt

```
